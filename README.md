# 赤壁之戰 3D 互動視覺化

Battle of Chibi — Interactive 3D Battle Visualization

## 功能

- **3D 地形場景**：長江、赤壁、烏林、夏口等地標
- **三軍陣營**：曹操（藍）、孫權（紅）、劉備（綠）
- **70+ 艘船艦**：各軍軍旗清晰可辨（曹 / 孫 / 劉）
- **12 位將領**：曹操、周瑜、劉備、諸葛亮、黃蓋、龐統、關羽等
- **時間軸控制**：拖曳/播放/暫停，完整呈現戰爭進程
- **20 個歷史事件**：從曹操南下到三足鼎立
- **動態特效**：火攻粒子、煙霧、箭雨、閃電、天氣變化
- **軍勢統計**：實時兵力、戰力變化
- **小地圖**：即時戰場縮略圖
- **自由攝影機**：左鍵旋轉、右鍵平移、滾輪縮放
- **自動運鏡**：隨戰役階段自動切換視角
- **將領互動**：點擊將領查看詳細資訊

## 操作

| 操作 | 功能 |
|------|------|
| 左鍵拖曳 | 旋轉視角 |
| 右鍵拖曳 | 平移場景 |
| 滾輪 | 縮放 |
| 空白鍵 | 播放/暫停 |
| 左右方向鍵 | 時間軸快進/快退 |
| 點擊將領 | 查看資訊 |

## 技術

- Three.js 0.163 (r163)
- 單一 HTML 檔案，無需後端
- CDN 載入，支援 GitHub Pages
