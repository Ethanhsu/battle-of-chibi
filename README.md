# 赤壁之戰 - 三國鼎立

赤壁之戰 3D 互動視覺化 — 電視特別節目級別的戰役重現。

## 功能特色

- **3D 地形場景**：長江、赤壁山、烏林、夏口、荊州、華容道
- **三軍艦隊**：曹操 48 艘（藍）、孫權 25 艘（紅）、劉備 12 艘（綠），共 85 艘船艦
- **10 位將領**：曹操、周瑜、劉備、諸葛亮、黃蓋、龐統、關羽、趙雲、魯肅、許褚
- **電影運鏡**：10 個專業攝影機角度，隨戰役階段自動切換，含平滑轉場
- **時間軸**：18 個歷史事件，從曹操南下到天下三分
- **計策特效**：連環計、苦肉計、草船借箭、借東風、火攻 — 大型彈窗提示
- **動態特效**：火攻粒子系統、煙霧、閃電、波浪隨風力變化
- **軍勢統計**：實時兵力、戰力變化面板
- **小地圖**：即時戰場縮略圖 + 相機位置
- **音效系統**：Web Audio API 合成 — 戰鼓、号角、火攻爆裂、風聲、背景配樂
- **全中文介面**

## 操作方式

| 操作 | 功能 |
|------|------|
| 滑鼠左鍵拖曳 | 旋轉視角 |
| 滑鼠右鍵拖曳 | 平移場景 |
| 滾輪 | 縮放 |
| 空白鍵 | 播放 / 暫停 |
| 左右方向鍵 | 時間軸快進 / 快退 |
| 點擊將領標記 | 聚焦到該將領 |
| 滑鼠操作後 | 4 秒後自動恢復電影運鏡 |

## 技術

- Three.js r163 (CDN)
- Web Audio API (程式合成音效)
- 單一 HTML 檔案，無需後端
- 低耗能模式 (low-power GPU)
