# 赤壁之戰 - 三國鼎立

赤壁之戰 3D 互動戰鬥視覺化 — 歷史修正版。

## 歷史考據

- 曹操佔荊州後順長江東下，主力駐紮烏林（北岸）
- 劉備長坂坡敗後退至夏口（漢水入江口）
- 孫權遣周瑜率水軍三萬東下，於赤壁（南岸）與劉備會合
- 赤壁在烏林對岸，隔長江相望
- 火攻為水戰，黃蓋率火船衝入曹軍船陣
- 曹軍敗後由陸路經華容道撤退（泥濘小路，不在河上）
- 關羽受命駐守華容道（戰後才出現）

## 功能

- 正確地理佈局：長江、烏林（北岸曹軍）、赤壁（南岸聯軍）、華容道（陸路）
- 三軍艦隊 82 艘船艦，依歷史路線移動
- 10 位將領：曹操、周瑜、劉備、諸葛亮、黃蓋、龐統、魯肅、許褚、關羽（戰後登場）
- 21 個歷史事件，按時間軸展開
- 21 個電影鏡頭
- 火攻特效：爆炸粒子、火焰、煙霧、火光照明
- 隔江射擊箭矢
- 全中文介面
- Web Audio 音效

- 滑鼠左鍵旋轉 · 右鍵平移 · 滾輪縮放
- 空白鍵播放/暫停 · 方向鍵快進快退
- 點擊將領聚焦
